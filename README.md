# bootstrap-next

Quickly and easily bootstrap a Next.js project.

## Technologies

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [ESLint](https://eslint.org/)
- [Yarn](https://yarnpkg.com/)

## File Structure

```
project/
├── package.json
├── .gitignore
├── README.md
├── pages/
│   ├── index.js
├── components/
├── lib/
├── static/
```

## Usage

```
$ yarn global add bootstrap-next
$ mkdir project
$ cd project
$ bootstrap-next
$ yarn dev
```
